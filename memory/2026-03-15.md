## 下午/晚上追加（2026-03-15）

### 解决的问题

1. **视频缩略图 + 列表预览** - NoteRow 支持视频缩略图、点击全屏播放、时长角标
2. **标签展示优化** - 每个标签独立胶囊，最多3个+超出显示 +N
3. **视频数据未保存 bug** - onSave 签名补 videoData + videoDuration 参数
4. **媒体选择 UI 重设计** - 统一工具栏（拍照｜相册｜录视频），去掉奇怪的三按钮竖排
5. **照片+视频混合** - 录完视频可继续加照片，类型自动变 .mixed
6. **相册选视频** - CustomPhotoPickerView 支持图片+视频混合，PHAsset duration 限制大视频
7. **视频选择闪退（iOS 26 Swift 6 actor 隔离）** - 根因：PHImageManager 后台线程回调 + @MainActor 冲突。修法：nonisolated + withCheckedContinuation
8. **视频录制改5秒** - maxSeconds = 5，提示更新
9. **录完没自动关** - capturedDuration 在 finishRecording 时锁定，避免 delegate 读到竞争值 0
10. **录制视频存相册** - PHPhotoLibrary.shared().performChanges
11. **键盘收不回去** - scrollDismissesKeyboard(.interactively)
12. **编辑页两排按钮** - 编辑模式隐藏 mediaActionBar

### 关键教训

**Swift 6 / iOS 26 新坑：**
- PHImageManager 的 completion handler 在后台线程回调
- @MainActor 的 View 里调用它，Swift 6 会直接 crash（不是 warning！）
- 修法：confirmSelection 改 async Task，fetch 方法改 nonisolated
- **以后所有 PHImageManager / PHAssetResourceManager 回调必须用 nonisolated + continuation 桥接**

**流程教训：**
- 应该先开 SWIFT_STRICT_CONCURRENCY=complete 编译检查，让编译器发现 actor 问题
- 视频测试不能全靠手测，用 VideoRobustnessTests 覆盖后台线程安全、时长竞争、数据传递

- VideoFeatureTests2.swift - 19 个单元测试
- VideoRobustnessTests.swift - 15 个健壮性测试（含后台线程、竞争条件、端到端）

### 视频功能完整重写总结

**架构最终状态：**
- `MediaItem` 枚举统一管理图片+视频（最多9格）

- `CameraView.onSave` 签名：`(Data?, [Data]?, NoteType, String, String?, Double?, Double?, [String]?, [String]?, [String]?, [Double]?, [String]?) -> Void`

**VideoRecorderView 状态机：**
```swift
enum RecordState { case idle, recording, finishing, cancelled, done }
```
- 去掉 maxRecordedDuration，用整数 tickCount 计数（避免浮点累加误差）
- Timer 用 .common RunLoop mode（长按手势期间不会失效）
- delegate 看 state 判断意图

**VideoPlayerView 改用 URL：**
- `VideoPlayerView(videoURL: URL)` 直接播
- `VideoThumbnailView(videoURL: URL)` 直接取帧
- `VideoDataPlayerView` / `VideoDataThumbnailView` 包装旧 Data 兼容

**CameraView.stableCopy：**
- 选完视频后立即复制到 Documents/Videos/
- 防止 temp 文件被系统清理导致保存失败

**今天最大的教训：**
1. 打补丁不如重写，第一次没设计好要早重写
2. Timer 在 UILongPressGestureRecognizer 期间需要 .common mode
3. Swift 6 PHImageManager 回调必须用 nonisolated + continuation
4. 浮点累加（0.1 * 50 != 5.0）用整数计数器
5. 时长显示统一用 `max(1, Int(ceil(s)))`
