## 随手记 App - 用户动线图设计

### 项目背景
为随手记 App 创建完整的用户动线流程图，展示 5 大核心用户流程。

### 设计风格
- 简洁白色背景（#f5f5f5）
- 蓝色"拍照"按钮（#007aff）
- 绿色"写字"按钮（#34c759）
- 灰色取消按钮（#e5e5e5）
- 红色删除按钮（#ff3b30）
- 原生 iOS 风格卡片和弹窗

### 5 大核心用户流程

1. **📷 流程 1：拍照上传单张图片**
   - P1 首页 → P2 拍照选择 → P3 相机 → P2 预览 → P1 首页

2. **🖼️ 流程 2：从相册选择多张图片 (1-9 张)**
   - P1 首页 → P2 拍照选择 → P4 相册 → P2 预览 → P1 首页

3. **✏️ 流程 3：新建纯文字笔记**
   - P1 首页 → P5 文字编辑 → P1 首页

4. **📋 流程 4：编辑已有笔记**
   - P1 首页 → P6 图片编辑 → P4 相册 → P6 编辑 → P1 首页

5. **🗑️ 流程 5：删除笔记**
   - P1 首页 → P7 删除确认 → P1 首页（卡片移除，下方自动上移）

### 9 个核心页面
- P1: 首页 (ContentView)
- P2: 拍照/相册选择页
- P3: 系统相机
- P4: 系统相册 (PhotosPicker)
- P5: 文字编辑页
- P6: 图片编辑页
- P7: 删除确认弹窗
- P8: 放弃确认弹窗
- P9: 上下文菜单
