## 🐛 问题描述

### 编译错误
```
Cannot call value of non-function type 'CGSize'
```

**位置：** `CameraView.swift` - `aspectFitSize` 函数

---

## 🔍 问题原因

**根本原因：** 参数名 `max` 与 Swift 内置函数 `max()` 命名冲突

**问题代码：**
```swift
private func aspectFitSize(_ size: CGSize, max: CGSize) -> CGSize {
    let widthRatio = size.width / max.width
    let heightRatio = size.height / max.height
    let ratio = max(widthRatio, heightRatio)  // ❌ 编译器混淆
}
```

**解释：**
- Swift 看到 `max(widthRatio, heightRatio)` 时
- 无法区分是调用内置函数 `max()` 还是使用参数 `max`
- 导致编译错误

---

## ✅ 修复方案

### 修改内容

**文件：** `CameraView.swift`

**修复 1：参数名改名**
```swift
// 修复前
private func aspectFitSize(_ size: CGSize, max: CGSize) -> CGSize

// 修复后
private func aspectFitSize(_ size: CGSize, maxSize: CGSize) -> CGSize
```

**修复 2：函数调用明确化**
```swift
// 修复前
let ratio = max(widthRatio, heightRatio)

// 修复后
let ratio = Swift.max(widthRatio, heightRatio)
```

**修复 3：调用处更新**
```swift
// 修复前
let size = aspectFitSize(image.size, max: maxSize)

// 修复后
let size = aspectFitSize(image.size, maxSize: maxSize)
```

---

## 📝 完整修复代码

```swift
/// 计算等比缩放后的尺寸
private func aspectFitSize(_ size: CGSize, maxSize: CGSize) -> CGSize {
    let widthRatio = size.width / maxSize.width
    let heightRatio = size.height / maxSize.height
    let ratio = Swift.max(widthRatio, heightRatio)
    
    if ratio > 1 {
        return CGSize(width: size.width / ratio, height: size.height / ratio)
    }
    return size
}
```

## 🎯 后续建议

### 代码规范
1. **避免使用保留字作为参数名**

   - 参数名应该清晰描述用途
   - 避免单字母或过于简短的名称
