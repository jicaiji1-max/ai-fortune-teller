关键数据流
① 新建笔记
ContentView→ CameraView→ onSave 回调→ Note(insert)→ modelContext.save()
② 编辑笔记 + 相册预选
NoteRow 点击→ CameraView(existingNote)→ assetIdentifiers→ CustomPhotoPickerView→ preselectedIds 高亮序号
③ 列表图片预览（全屏）
NoteRow tap→ PreviewManager.presentPhotos()→ ContentView overlay→ PhotoViewerView（全屏）
④ 编辑中预览
CameraView 缩略图 tap→ showPhotoPreview = true→ SwipeToDismissWrapper overlay→ PhotoViewerView
⑤ 视频存储路径
录制/相册选择→ stableCopy(Documents/Videos/)→ videoPaths 存绝对路径→ Note.videoURLs 计算属性还原
⚡ iOS 26 beta 已知问题及绕过方案
• NavigationStack 在 sheet 里黑屏 → CustomPhotoPickerView 改用自绘 Header Bar
• sheet-in-sheet / fullScreenCover 黑屏 → 改用顶层 ZStack overlay + PreviewManager
• SwiftUI TabView 第一页黑屏 → PhotoViewerView 完全重写为 UIKit UIPageViewController

⚡ Swift 6 并发注意事项
• PHImageManager / PHAssetResourceManager 回调在后台线程 → nonisolated + withCheckedContinuation 桥接
• SwiftData externalStorage fault 异步 → NoteRow 用 .task + retry 加载 photoData
• VideoRecorderView 状态机用 enum RecordState 避免竞争条件
