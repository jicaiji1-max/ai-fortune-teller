## 🔧 详细改进

### 1. 性能优化：DateFormatter 重复创建 ✅

**问题：** 每次渲染都创建新的 `DateFormatter`，性能开销大

**修复：**
- `NoteRow.swift`: 使用 static `timeFormatter`
- `ContentView.swift`: 使用 static `groupHeaderFormatter`

**影响：** 减少 CPU 占用，提升滚动流畅度

---

### 2. 性能优化：图片未压缩存储 ✅

**问题：** 直接存储原始图片数据，导致数据库膨胀

**修复：**
- `CameraView.swift`: 保存前压缩图片（quality: 0.7）
- 使用 `jpegData(compressionQuality: 0.7)` 压缩

**影响：** 减少存储空间约 50-70%

---

### 3. 性能优化：外部存储 ✅

**问题：** 大文件存在 SQLite 中会导致数据库变慢

**修复：**
- `Note.swift`: 添加 `@Attribute(.externalStorage)` 到 `photoData`

**影响：** SwiftData 自动将大文件存到文件系统

---

### 4. 功能增强：滑动删除 ✅

**问题：** 无法删除记录

**修复：**
- `ContentView.swift`: 添加 `.swipeActions` 和删除按钮
- 使用 `withAnimation` 平滑动画

**影响：** 用户可以左滑删除记录

---

### 5. UX 改进：放弃编辑确认 ✅

**问题：** 点击取消会丢失已输入内容，无任何提示

**修复：**
- `TextEditorView.swift`: 添加 `showDiscardAlert`
- 内容不为空时弹出确认对话框

**影响：** 防止误操作丢失内容

---

### 6. UX 改进：图片加载进度 ✅

**问题：** 加载大图片时无任何反馈

**修复：**
- `CameraView.swift`: 添加 `isLoadingImage` 状态
- 显示 `ProgressView` 加载指示器

**影响：** 用户体验更好
