## 🎨 多图展示方案

### 方案 A：网格拼贴（推荐）⭐

```
┌─────────────────────────┐
│  今天 11:50             │
├─────────────────────────┤
│ ┌──────┐ ┌──────┐       │
│ │ 图 1  │ │ 图 2  │       │
│ │      │ │      │       │
│ ├──────┤ ├──────┤  ...  │
│ │ 图 3  │ │ +5   │       │
│ └──────┘ └──────┘       │
│                         │
│ 很好吃呀，今天的酸菜   │
│ 鱼特别入味！            │
└─────────────────────────┘
```

**布局规则：**
| 图片数量 | 布局 |
|---------|------|
| 1 张 | 单张大图（当前样式） |
| 2 张 | 左右并排 |
| 3 张 | 左 1 大 + 右 2 小 |
| 4 张 | 2x2 网格 |
| 5-9 张 | 3x3 网格，最后一张显示"+N" |

**优点：**
- ✅ 信息密度高
- ✅ 一眼看到多张图
- ✅ 符合主流设计（微信朋友圈、小红书）

---

### 方案 B：轮播图

```
┌─────────────────────────┐
│  今天 11:50        ●○○  │
├─────────────────────────┤
│                         │
│    ┌──────────┐         │
│    │          │         │
│    │  图 1     │         │
│    │          │         │
│    └──────────┘         │
│                         │
│ 很好吃呀...             │
└─────────────────────────┘
```

**特点：**
- 左右滑动切换图片
- 底部圆点指示器
- 一次只看一张图

**优点：**
- ✅ 每张图都清晰展示
- ✅ 适合图片质量高的场景

**缺点：**
- ❌ 需要滑动才能看到全部
- ❌ 信息密度低

---

### 方案 C：首图 + 角标

```
┌─────────────────────────┐
│  今天 11:50             │
├─────────────────────────┤
│ ┌──────────────────┐    │
│ │                  │    │
│ │     首图         │ 3  │
│ │                  │    │
│ └──────────────────┘    │
│                         │
│ 很好吃呀...             │
└─────────────────────────┘
```

**特点：**
- 只显示第一张图
- 右上角显示数量角标

**优点：**
- ✅ 简洁，和单图一致
- ✅ 卡片高度统一

**缺点：**
- ❌ 看不到其他图
- ❌ 需要点击进入详情

## ✏️ 编辑功能设计

### 编辑入口

#### 方案 1：长按菜单
```
长按记录卡片
    ↓
弹出菜单：
- 📝 编辑
- 📤 导出到相册
- 🗑️ 删除
```

#### 方案 2：左滑操作
```
← 滑动卡片
    ↓
显示按钮：
[编辑] [删除]
```

#### 方案 3：点击进入详情
```
点击卡片
    ↓
进入详情页
    ↓
右上角 [编辑] 按钮
```

**推荐：方案 1 + 方案 3 组合**
- 长按 → 快速编辑
- 点击 → 查看详情
